# 运维

欢迎来到运维技术教程！本模块聚焦于现代云原生技术栈，帮助开发者掌握生产环境必备的运维技能。

## 教程列表

### Kubernetes

从零开始学习 Kubernetes（K8s），掌握容器编排技术。

- **适合人群**: 对容器和编排技术零基础的开发者
- **学习时长**: 约 8-12 小时
- **前置知识**: 基本命令行操作能力

[开始学习 Kubernetes](/ops/kubernetes/)

## 学习建议

1. **循序渐进**: 建议按照教程顺序学习，每个章节都建立在前一章的基础上
2. **动手实践**: 每个实践章节都提供可直接执行的命令，建议边学边做
3. **善用图表**: 复杂概念都配有 Mermaid 图表，帮助建立直观认知
