# 工具函数

Python 标准库提供了多种通用工具模块，覆盖哈希、编码、拷贝和上下文管理等常见需求。

| 模块 | 说明 |
|------|------|
| [hashlib 哈希](./hashlib.md) | MD5、SHA 系列等安全哈希算法 |
| [base64 编码](./base64.md) | Base64/Base32/Base16 编解码 |
| [copy 拷贝](./copy.md) | 浅拷贝和深拷贝工具 |
| [contextlib 上下文](./contextlib.md) | 上下文管理器工具和装饰器 |
