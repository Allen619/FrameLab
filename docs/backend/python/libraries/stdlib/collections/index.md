# 数据结构增强

Python 标准库提供了多个增强数据结构处理能力的模块，包括高性能容器、迭代工具和函数式编程工具。

| 模块 | 说明 |
|------|------|
| [collections 容器](./collections.md) | Counter、defaultdict、deque 等高性能容器 |
| [itertools 迭代器](./itertools.md) | 高效迭代器工具：排列组合、无限序列、链式迭代 |
| [functools 函数工具](./functools.md) | 缓存装饰器、偏函数、reduce 等函数式编程工具 |
| [enum 枚举](./enum.md) | 枚举类型定义与使用 |
