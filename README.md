# docs-site

学习文档整合站点 - 前端、后端、AI

## 内容

- **前端开发** - 前端技术学习资料
- **后端开发** - Python 开发教程，专为前端开发者设计
- **AI 技术** - LangChain 1.0 中文教程

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- [Mermaid](https://mermaid.js.org/) - 图表支持
